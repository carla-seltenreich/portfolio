Projetos

Durante a disciplina de Aplicações Web - Client Side participei de 3 projetos que foram importantes para o aprendizado adquiro até o momento. São eles a criação de paginas web, obedecendo fielmente requisitos especificados pelo cliente.

PROJETO 1 - Olaria Alegria O cliente proprietário precisava difundir seu estabelecimento e seus produtos no meio digital. Nos foi solicitado um site simples, onde o cliente encontra de forma fácil os produtos produzidos e comercializados pela empresa.

PROJETO 2 - Pet&Short Neste projeto o cliente foi bem específico de cada detalhe em seu site, a fim de utilizar como uma das principais formas de divulgação dos serviços prestados. Com abas onde se encontrava explicações detalhadas de cada serviço e página direta para contatar a empresa.
